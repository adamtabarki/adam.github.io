qu'est ce que le code morce ?

Le code Morse est un schéma de codage de caractères qui permet aux opérateurs
d'envoyer des messages en utilisant une série d'impulsions électriques représentées
sous la forme d'impulsions courtes ou longues : en d'autres termes, des points et des tirets.

Qui a inventé le code Morse ?

Samuel F. B. Morse et son assistant Alfred Vail sont connus pour être les inventeurs de ce code
auquel le scientifique a donné son nom.

Quand le code Morse a-t-il été inventé ?

Le code Morse a été développé dans les années 1830, puis perfectionné dans les années 1840.

Quel était le premier message envoyé en code Morse ?

« What hath God wrought » (en français : Ce que Dieu a forgé) est le premier message officiel envoyé
par Samuel F.B. Morse le 24 Mai 1844 pour ouvrir la ligne télégraphique Baltimore - Washington.

À quoi le code Morse sert-il ?

Dans le passé, le code Morse bénéficiait d'une large utilisation, notamment dans l'armée. Bien que son
domaine d'utilisation soit aujourd'hui limité, il est toujours employé dans l'aviation, les activités de
radio amateur et les technologies d'assistance (TA).

code Morse.
